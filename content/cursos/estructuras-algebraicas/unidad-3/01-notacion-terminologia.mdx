# Notación y Terminología

A medida que profundizamos en la teoría de grupos, es útil establecer ciertas convenciones de notación para agilizar la escritura.

## Notación Multiplicativa y Aditiva

En la mayoría de los casos generales o abstractos, omitimos el símbolo de la operación binaria $*$ y escribimos $a * b$ simplemente como la yuxtaposición $ab$. Esto se conoce como **notación multiplicativa**. En este contexto, el elemento identidad se denota frecuentemente por $e$ o $1$, y el inverso de $a$ como $a^{-1}$.

Para grupos que son estrictamente **abelianos** (como los enteros o los reales bajo la suma), se suele emplear la **notación aditiva**. Aquí la operación se escribe como $+$, la identidad como $0$, y el inverso del elemento $a$ se denota como $-a$.

## Exponenciación

Bajo la notación multiplicativa, la aplicación repetida de una operación sobre un mismo elemento se abrevia mediante exponentes:

<div className="text-center font-semibold my-4 px-4">
  $a^n = a \cdot a \cdot \dots \cdot a$ (n veces)
</div>

También definimos convencionalmente $a^0 = e$ y $a^{-n} = (a^{-1})^n$. Las leyes habituales de los exponentes aplican: $a^m a^n = a^{m+n}$ y $(a^m)^n = a^{mn}$.

## Orden de un Grupo y Orden de un Elemento

- **Orden de un grupo $|G|$:** Es el número de elementos en el conjunto $G$.
- **Orden de un elemento $|a|$:** Es el menor entero positivo $n$ tal que $a^n = e$. Si tal entero no existe, se dice que el elemento tiene orden infinito.
